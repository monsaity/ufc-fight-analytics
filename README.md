# UFC Fight Analytics

## Цель проекта

Провести анализ данных боёв UFC с использованием Python, SQL и статистических методов, выявить факторы, связанные с победой бойца, а также построить модель предсказания исхода боя.

## Используемые технологии

- Python
- pandas
- numpy
- SQL
- SQLite
- matplotlib
- scipy
- scikit-learn

## План работы

1. Сбор данных
2. Очистка и подготовка данных
3. Создание базы данных SQL
4. Исследовательский анализ данных
5. Статистическая проверка гипотез
6. Построение модели предсказания
7. Итоговые выводы
